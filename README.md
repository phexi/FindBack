I built an Android app to help my parents remember where they put things
My mom constantly forgets where she puts important stuff like medicine, documents, keys, chargers, etc.
So I started building a simple Android app for her.
The idea is very simple:
Take a photo of the item
Take photos of where it’s stored (room → cabinet → drawer → exact spot)
Later, just search the item name to find it again
I’m trying to make it especially easy for seniors:
Large buttons
Minimal UI
Photo-based memory instead of complicated folders/tags
Planning to add voice search later
Still an early version, but I’d really love feedback.
Would something like this be useful for your parents or for yourself?
